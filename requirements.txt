# Use the `--only-binary=:all:` flag to force pip to use prebuilt wheels.
# Without it, PyYAML 6.0 fails to build on Python 3.14 (Cython compat issue:

# pinned pydantic-core 2.14.1 has no wheel for Python 3.14. The flag pulls
# PyYAML 6.0.3 and the latest pydantic-core wheel instead.
#
# Pydantic and python-dotenv use >= because older exact pins (2.5.0, 1.0.1)
# lack Python 3.14 wheels for their transitive deps. The codebase works
# fine with the newer versions — verified by `run.py --test`.
